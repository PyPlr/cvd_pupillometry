Citation
========

If you use *PyPlr* in a scientific publication we would appreciate citations to the published software and the following paper:

.. code-block:: bibtex

    @software{PyPlr2021,
      author         = {Martin, Joel T and Spitschan, Manuel},
      title          = {PyPlr (version 1.0.2)},
      month          = {August},
      year           = {2021},
      publisher      = {Zenodo},
      version        = {v1.0.2},
      doi            = {10.5281/zenodo.5749833},
      url            = {https://doi.org/10.5281/zenodo.5749833}
    }

    @article{Martin2021,
      author         = {Martin, Joel T and Pinto, Joana and Bulte, Daniel P and Spitschan, Manuel},
      title          = {PyPlr: A versatile, integrated system of hardware and software for researching the human pupillary light reflex},
      year           = {2021},
      doi            = {https://doi.org/10.3758/s13428-021-01759-3},
      publisher      = {Springer},
      URL            = {https://link.springer.com/article/10.3758%2Fs13428-021-01759-3},
      eprint         = {https://link.springer.com/content/pdf/10.3758/s13428-021-01759-3.pdf},
      journal        = {Behavior Research Methods}
    }
